# Website Undangan Pernikahan Rizky & Dinda

Website undangan pernikahan digital untuk Rizky & Dinda dengan tampilan sakral dan fitur lengkap.

## Fitur

1. **Tampilan Responsif** - Dapat diakses dari desktop, tablet, dan smartphone
2. **Dua Galeri Foto** 
   - Galeri pertama pada halaman pembuka
   - Galeri kedua pada halaman undangan utama
3. **Background Music** - Musik latar dengan kontrol ON/OFF
4. **Hitung Mundur** - Menuju tanggal pernikahan
5. **Informasi Donasi** - Rekening BCA untuk blessing
6. **Lokasi Acara** - Dengan peta Google Maps
7. **Buku Tamu Digital** - Ucapan dan doa untuk pengantin
8. **Navigasi Mudah** - Menu floating untuk akses cepat
